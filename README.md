# 🎮 Color Memory Game

A fun memory game developed using **Python** and **Pygame**. The game generates an increasingly complex sequence of colors that the player must memorize and repeat correctly. The difficulty increases with each successful round.

## Features
- Interactive graphical interface using Pygame
- Random color sequence generation
- Increasing difficulty with each level
- Score tracking
- Leaderboard support

## Technologies Used
- Python
- Pygame

## How to Run

1. Clone the repository:
   ```bash
   git clone https://github.com/Joshitha538/color-memory-game.git
   ```

2. Install Pygame:
   ```bash
   pip install pygame
   ```

3. Run the game:
   ```bash
   python color_memory_game.py
   ```
